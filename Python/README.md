**FareCalc - Travel Fare Calculator (Python)**

**Project Description**

This project is a simple Fare Calculation System built using Python.
It calculates the estimated ride fare based on distance, vehicle type, and time of the day (surge pricing).

**Objective**

To simulate a basic backend system used by ride-sharing platforms to calculate dynamic pricing.

**Features**
Calculates fare based on distance (per km rate)

Supports multiple vehicle types:
Economy, 
Premium, 
SUV

Applies surge pricing (1.5x) during peak hours (5 PM – 8 PM)

Handles invalid vehicle input

Displays a formatted ride receipt

**Concepts Used**

Python Basics, 
Functions, 
Dictionary (for storing rates), 
Conditional Statements (if-else), 
Exception Handling (try-except), 
User Input Handling

**Technologies Used**

Python 3.x
