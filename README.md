**Password Validator (Java)**
**Project Description**

This is a simple Java-based Password Validator program that checks whether a given password meets basic security requirements. It ensures that the password is strong enough by validating specific conditions.

**Features**
1)Checks if password length is at least 8 characters
2)Ensures presence of at least:
a)One uppercase letter
b)One digit
3)Displays clear messages for missing conditions
4)User-friendly console interaction

**Technologies Used**
Java
Scanner class (for user input)
