**Password Validator (Java)**

**Project Description**

This project is a simple Password Validation System developed using Core Java.
It checks whether a user-entered password satisfies basic security requirements and provides feedback if it does not.

**Objective**

To ensure that users create strong passwords by validating them against predefined rules.

**Features**

Checks minimum password length (at least 8 characters)

Verifies presence of:

At least one uppercase letter

At least one digit

Provides clear feedback messages for invalid passwords

Simple console-based interaction

**Concepts Used**

Java Basics

Conditional Statements (if-else)

Loops (for loop)

String Handling

Methods (Function creation)

Scanner class for user input

**Technologies Used**
